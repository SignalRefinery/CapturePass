# CapturePass Brand Assets

Put all final CapturePass artwork in this folder so the brand files stay centralized.

Preferred filenames:

- `CapturePass Logo.png`
- `CapturePass Wordmark.png`
- `Capturepass Logo with Wordmark.png`
- `Capturepass Logo with Wordmark and Tagline.png`

Optional subfolders:

- `marketing/`
- `mockups/`

Keep new brand artwork here instead of spreading assets across the repo.
